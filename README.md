# Assistant liste

This assistant python plugin allows managing lists of entities that can be of different types

## Pré-requis

Mandatory : The "plugin\_maitre" must be installed.
Link to the plugin maître : [maitre-qgis-plugin sur GitHub](https://github.com/IGNF/maitre-qgis-plugin)
qgisMinimumVersion=3.4

## Fonctionnalités

French :
* Création de listes d'objets suivant différents modes​
* Import / Export de listes​
* Suppression de listes d'objets suivant différents modes​
* Lien pour ouvrir la table attributaires des entités de la liste

English : 
* Creation of object lists using different operation modes
* Import / Export of lists
* Deletion of object lists using different operation modes
* Link to open the attribute table of the entities in the list
